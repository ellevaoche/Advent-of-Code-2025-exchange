
Download input-file from `https://adventofcode.com/2025/day/1/input` as `input.txt`

Compile:
```
javac aoc/*.java
```

Run Test:
```
java -cp . aoc.Test
```

Run:
```
java -cp . aoc.Main < input.txt
```

```
java -cp . aoc.Test_0x434C49434B
```

Run Part 2:

java -cp . aoc.Main_0x434C49434B < input.txt
